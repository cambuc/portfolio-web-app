# Cameron Buchanan — Portfolio

Personal portfolio site built with React and Vite. Hosted on GitHub Pages.

**Live site:** https://cambuc.github.io/portfolio-web-app/

## Sections

- **Hero** — intro and contact links
- **Skills** — technical skills
- **Experience** — work history
- **Projects** — live GitHub repositories via the GitHub API

## Tech Stack

- React 18 + Vite
- Plain CSS with custom properties
- GitHub REST API
- gh-pages for deployment

## Development

```bash
npm install
npm run dev
```

## Deploy

```bash
npm run deploy
```

Builds the project and pushes to the `gh-pages` branch.
